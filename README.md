# ECN/EDN French Medical Textbook Research

Research project to find all downloadable ECN/EDN reference PDFs and textbooks
from French publishers for all medical specialties.

## Publishers Searched
- Elsevier-Masson (Collection des Collèges)
- Vernazobres-Grégo (Référentiels des Collèges)
- S-Éditions (Éditions S)
- Decarre
- Studyrama

## Files
- `scraper.py` - Main scraping script
- `discovery_results.json` - Phase 1: URL discovery
- `deep_results.json` - Phase 2: Detailed product scraping
- `final_summary.json` - Phase 3: Summary report
